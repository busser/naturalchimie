# Naturalchimie clone

This repo holds a specification (and eventual implementation) for a faithful
clone of the classic Flash puzzle game *Naturalchimie* by Motion Twin.

Always read the overview before working on this project:

@docs/00-overview.md

The full spec lives in `docs/`, split across multiple files
(`00-overview.md` through `09-responsive-layout.md`). Read the specific
file relevant to the current task; don't load them all by default.
