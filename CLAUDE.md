# Naturalchimie clone

This repo holds a specification (and eventual implementation) for a faithful
clone of the classic Flash puzzle game *Naturalchimie* by Motion Twin.

Always read the overview before working on this project:

@docs/00-overview.md

The full spec lives in `docs/`, split across seven files (`00-overview.md`
through `06-acceptance-tests.md`). Read the specific file relevant to the
current task; don't load all seven by default.
